# Elixir Playground

This is just an area for some playing with Elixir. So far it's for getting a good handle on testing and GenServer.

I've implemented most of the code in [`test/`](test/) directly for quick iteration and testing.

## Usage

```sh
script/setup
```
```sh
script/test
```
